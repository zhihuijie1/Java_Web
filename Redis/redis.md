后端启动redis的指令

```shell
redis-server /home/terrychen/redis-conf
```

用这个指令来擦看redis运行的进程

```
ps -ef | grep redis
```

后台启动可以是用客户端来连接我们的redis

```
redis-cli
```

```
exit //退出
```

redis关闭的指令

```
redis-cli shutdown
```
